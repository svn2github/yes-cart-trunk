If you're seeing this page via a web browser, it means you've setup Yes Cart demo successfully. Congratulations!

This is the default page of demo server. If you see this then tomcat had successfully started.

Now you can get started with Yes Cart

  • YUM (Yes cart Update Manager) is here: https://localhost:8443/yes-manager/
  • YeS (Yes cart Storefront) is here: http://localhost:8080/yes-shop/
  • You can also use Tomcat's monitoring tools (see Tomcat admin menu).
    The username is "yc" and password is "yc"

This demo server contains only initial data - just enough to see it alive. This is done purposefully to give you the idea of how it all works. At the moment there is a single shop instance configured with no data.
Next step is to populate this shop instance with catalog, products, inventory and price information.
You can do this either by manually entering the data in YUM, or alternatively you can use YUM to import our sample data from the System section.

We strongly recommend going through our online documentation prior to making any changes. Once you understand the basic idea you'll be able to do all sorts of things with ease, otherwise you may face some unnecessary challenges.

Sample data is kindly provided by IceCat and is bundled with this demo server.
It is located in:
$SERVER_HOME/icecat-demo/
where $SERVER_HOME is the name of directory where you have unzipped the demo server.
There are two import files:

  • import-EN,RU.zip - which contains all data, and
  • import-EN,RU-imh.zip - which contains product images
Once you import these two you should have more than plenty to play around with Yes Cart.
